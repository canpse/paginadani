# Assets Visuais

Estrutura sugerida para o tema diegetico (caderno vintage).

## Pastas
- `bordas/svg`: molduras vetoriais, cantos ornamentais, divisores
- `bordas/png`: bordas com textura/transparencia
- `texturas/papel`: papel envelhecido, fibras, ruído fino
- `texturas/tinta`: pincel, manchas, respingos
- `texturas/grunge`: desgaste, poeira, riscos
- `icones/ui`: icones de acao (editar, salvar, filtro)
- `icones/simbolos`: icones narrativos (pato, pena, selo)
- `decorativos/selos`: carimbos e selos de cera
- `decorativos/ornamentos`: arabescos, florais, elementos de canto
- `decorativos/fitas`: fitas/marcadores de pagina
- `decorativos/manchas`: overlays leves para camadas
- `posts/capas`: imagens de capa dos posts
- `posts/thumbs`: miniaturas para lista/feed
- `fundos/hero`: fundos principais de topo
- `fundos/secundarios`: variações de fundo para seções

## Convenções de nome
- use minúsculas
- sem espaços e sem acentos
- formato recomendado: `tipo-descricao-01.ext`
- exemplo: `selo-cera-01.png`, `textura-papel-03.webp`
